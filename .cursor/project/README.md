# Project Overlay Files

This directory adapts the Cursor workflow template to this repository.

| File | Purpose |
|------|---------|
| `PROJECT.md` | Project identity, repo map, enabled packs, constraints. |
| `STRUCTURE.md` | Repository-specific folder, module, ownership, and pattern map. |
| `WORKFLOW.md` | Phase and task plan consumed by workflow execution. |
| `TOOLING.md` | Build, test, lint, format, and run commands. |
| `RULES.md` | Repository-specific hard and soft rules. |
| `CODING_CONVENTIONS.md` | Concrete coding style for this project. |
| `LEARNED.md` | Repeated project-specific patterns discovered during work. |
| `PROGRESS.md` | Human-readable workflow status. |
| `EVENTS.jsonl` | Append-only event journal, created when workflow starts. |
